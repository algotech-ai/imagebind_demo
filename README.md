# ImageBind 演示项目

## 项目描述
这是一个使用 ImageBind 和 FFmpeg 的演示项目，用于展示多模态（图像、音频、文本等）嵌入生成能力。

## 环境要求
- Linux 系统（推荐 Ubuntu 20.04+）
- Python 3.8+
- FFmpeg

## 安装指南

### 1. 安装系统依赖
```bash
apt-get -y update && apt-get install -y ffmpeg
```

### 2. 安装pip
pip install -r requirements.txt

### 3. 运行
python imagebind_demo.py
